Documentation des fonctions membres

◆ create_topic()

create_topic ( )

Traite la création d'un nouveau topic. Valide les entrées et enregistre le topic si valide.

◆ delete()

delete ( )

Permet la suppression d'un topic spécifié par son ID. Redirection vers la page principale du forum après suppression.

◆ home()

home ( )

Affiche la page principale du forum avec la liste des topics. Permet une recherche par mots-clés.

◆ manage()

manage ( )

Affiche une page de gestion pour les topics, accessible uniquement par les modérateurs et administrateurs. Permet la visualisation des topics pour éventuelle édition ou suppression.

◆ topic()

topic ( )

Affiche un topic spécifique avec ses détails et commentaires. Permet également la suppression de commentaires et la publication de nouveaux commentaires si l'utilisateur est connecté.
