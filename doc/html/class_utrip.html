Description détaillée

Représente un Utrip, définissant les caractéristiques et les fonctionnalités d'un voyage ou d'une expérience partagée sur la plateforme. Cette classe hérite de la classe Entity, permettant ainsi une gestion standardisée des entités, incluant l'hydratation à partir d'un tableau de données.
