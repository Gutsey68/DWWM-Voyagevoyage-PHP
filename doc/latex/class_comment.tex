\doxysubsection{Description détaillée}
Classe entité représentant un commentaire dans l\textquotesingle{}application. Elle hérite de la classe \doxylink{class_entity}{Entity} pour une intégration facile avec le système de base de données. Permet la gestion des attributs d\textquotesingle{}un commentaire comme son contenu, date, image associée, et le créateur.
