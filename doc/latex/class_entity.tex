{\footnotesize\ttfamily hydrate (\begin{DoxyParamCaption}\item[{}]{\$arr\+Data }\end{DoxyParamCaption})}

Hydrate l\textquotesingle{}objet avec les données fournies dans le tableau. Parcoure chaque clé du tableau et, si un setter correspondant existe dans l\textquotesingle{}objet, appelle ce setter avec la valeur associée. Les clés du tableau doivent correspondre aux noms des attributs de l\textquotesingle{}objet, préfixés et formatés pour correspondre à la convention des setters.


\begin{DoxyParams}[1]{Paramètres}
