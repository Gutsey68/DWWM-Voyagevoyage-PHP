\doxysubsection{Description détaillée}
Classe de base pour la connexion à la base de données. Elle établit une connexion avec la base de données via PDO et configure les options de base pour l\textquotesingle{}encodage et la gestion des erreurs. Cette classe est destinée à être héritée par d\textquotesingle{}autres modèles spécifiques qui requièrent une connexion à la base de données.
