\doxysubsection{Description détaillée}
Représente un \doxylink{class_utrip}{Utrip}, définissant les caractéristiques et les fonctionnalités d\textquotesingle{}un voyage ou d\textquotesingle{}une expérience partagée sur la plateforme. Cette classe hérite de la classe \doxylink{class_entity}{Entity}, permettant ainsi une gestion standardisée des entités, incluant l\textquotesingle{}hydratation à partir d\textquotesingle{}un tableau de données.
